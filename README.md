# Перевод текста в iCal

Константы:
```js
const TIMEZONE = '+04:00';
const TZID = "Europe/Samara";
const STARTDATE = [2021, 02, 01];
const ENDPERIOD = timeStamp(new Date('2021-06-31'));
```

Запуск:
```ps1
node .\index.js\ | Out-File .\output.ics
```
